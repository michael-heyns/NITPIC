
An archive of the original NITPIC commercial compiler from the 90s

## History

- Mixed C/BASIC syntax designed specifically for PIC microcontrollers
- True bit-level boolean type (8 named booleans used 1 byte)
- Bit addressing syntax: `MyByte:3` references bit 3
- Full MPLAB integration with debugging
- Extremely compact binary code

## Technical

- Written in C
- Originally compiled with Borland C++
- Last compiled 2013 with bcc32
- Still runs on Windows 11

## Status

Historical artifact. Not maintained. Shared for preservation.
